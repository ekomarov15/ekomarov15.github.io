# Конструктор КИМов ОГЭ и ЕГЭ по русскому языку

Учебный проект для подготовки к экзаменам.
Позволяет:
- выбирать типы заданий
- настраивать количество
- автоматически проверять тестовую часть
- сохранять результаты в браузере

## Публикация на GitHub Pages
1. Создайте репозиторий
2. Загрузите файлы
3. Settings → Pages → main / root

## Важно
Проект не является официальным ресурсом ФИПИ или sdamgia.ru.
Все задания являются авторскими примерами.
